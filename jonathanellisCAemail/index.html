How to install on iPhone / iPad

  1. Tap the green Install profile button above. iOS will say "This website is trying to download a configuration profile." Tap Allow.
  2. iOS shows: "Profile downloaded." Open the Settings app. At the very top (under your name) you'll see Profile Downloaded — tap it.
  3. Tap Install in the top-right. Enter your iPhone passcode if asked.
  4. iOS prompts for the mailbox password — type the one you set when creating the account in hq.jonathanellis.dev. Tap Next, then Done.
  5. Open Mail. The new account appears within ~5 seconds.

How to install on Mac

  1. Click the green Install profile button. The .mobileconfig downloads to your Downloads folder.
  2. Double-click the downloaded file. macOS opens System Settings → Privacy & Security → Profiles.
  3. Click the jonathanellis.ca Mail profile, then Install. Enter your Mac password.
  4. macOS prompts for the mailbox password — type it. Done.

Trouble connecting?

  • Try cellular data instead of WiFi. Some café/hotel/corporate WiFi blocks port 993.
  • If you previously had hello@jonathanellis.ca at GoDaddy / Workspace Email: delete that account first (Settings → Mail → Accounts → tap account → Delete) before installing the new profile.
  • Verify the password by logging into hq.jonathanellis.dev. Reset there if needed.
Privacy: the email server runs on Jonathan's own infrastructure (Hetzner Cloud, Nuremberg). Mail isn't filtered through GoDaddy / MXroute / Microsoft / Google. SMTP outbound goes through Brevo's authenticated relay so messages land in inboxes rather than spam.
