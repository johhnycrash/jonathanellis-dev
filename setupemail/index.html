What happens when you tap install

  1. iPhone / iPad: Settings opens, prompting you to review the profile. Tap Install in the top right. Enter your device passcode.
  2. Mac: System Settings opens to Profiles. Click the new profile, then click Install.
  3. You'll be asked for the mailbox password. Enter the one Jonathan gave you. (If you don't have it, email me.)
  4. Open Mail.app — the new account appears with all settings already correct.

To remove an account later

iPhone / iPad: Settings → General → VPN & Device Management → select the profile → Remove.

Mac: System Settings → Privacy & Security → Profiles → select → minus button.

Server hostname: use mail.jonathanellis.dev for @jonathanellis.dev addresses, and mail.jonathanellis.ca for @jonathanellis.ca addresses. The cert covers both, and DNS points both at the same server, so either works — but matching the domain to the hostname keeps things tidy.

Trouble connecting?

  • Try cellular data instead of WiFi. Some café/hotel WiFi networks block port 993.
  • If you previously trusted a self-signed cert: delete the account fully on iPhone, then re-install via the profile above. iOS caches old certs.
  • Check the password is exactly the one you were given (case-sensitive, no extra spaces).
  • If still stuck: email me and I'll reset.
